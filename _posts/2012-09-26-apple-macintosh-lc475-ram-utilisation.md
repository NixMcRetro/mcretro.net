Found that my RAM was all being eaten up by the system because 32-bit addressing was not turned on by default in System 7.5.3. I had 36MB of RAM, 30MB of it was allocated to the system software. Turned on 32-bit addressing via the memory control panel and it went right back to 2MB. I knew System 7 wasn't THAT hungry for RAM.

Found out about this thanks to [MicroMac](https://www.micromac.com/FAQ/faq_lc475_upgrade.html).
