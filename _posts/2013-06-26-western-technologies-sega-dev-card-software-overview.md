This gives a peek at the software running behind the Western Technologies Development Card for uploading (they call it downloading) from the PC to the Mega Drive. Works the same way as a Mega EverDrive over an LPT port instead of using an EEPROM! Beats having to burn the EPROMs with the game code every time you update code!
