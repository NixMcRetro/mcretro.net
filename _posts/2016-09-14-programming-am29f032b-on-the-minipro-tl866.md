> **"Hi, you can use the MBM29F033C from Fujitsu to write AM29F032B since the pinout is the same."**

While I haven't been able to test this first-hand, it should work right? I tend to use the Type II boards, not the Type III from [buyicnow.com](https://www.buyicnow.com/). So while I haven't been able to program using the MiniPro, the GQ-4X4 hasn't exploded yet.

So this should work with both AMD AM29F032 or ST M29F032D chips, in theory.
