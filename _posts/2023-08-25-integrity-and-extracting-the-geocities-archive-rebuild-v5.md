Another awesome thing about using tar files is that you can search for things in them and extract them based on wildcard searches. More on that [here](https://linuxize.com/post/how-to-create-and-extract-archives-using-the-tar-command-in-linux/). I appear to have lost the original link for the above screenshot and neither Google or the Duck is helping today.

While keeping the [original GeoCities data](https://wiki.archiveteam.org/index.php/GeoCities) from [archive.org](https://archive.org/details/archiveteam-geocities) My GeoCities rebuild was slapped into tar but not gunzipped (.gz) because I couldn't get it to split the files into 793 separate 1GB files. The following command was used on an Ubuntu 22.04 system.

 

```
# Create a archive (tar) and split into a few pieces to allow better checksumming (sha256).
cd /media/ubuntu/GC_2TB/
tar cvf - Rebuild_v5/ | split --bytes=1GB - "/media/ubuntu/GC_2TB/REBUILD/GeoCities_Rebuild.tar." -a 3 -d

```

and then we ran sha256 - all on a machine with ECC RAM.

 

```
# Generate a list of sha256 checksums for each part.
cd /media/ubuntu/GC_2TB/REBUILD/
sha256sum GeoCities_Rebuild.tar.* > GeoCities_Rebuild.sha256

```

We could then copy the data across the rickety, old network and verify the files had the same sha256 hashes.

 

```
# Checks all the files in the sha256 file for sha256 checksum matches - once on the Mac Pro SSD locally
cd /media/ubuntu/GC_2TB/REBUILD/
sha256sum -c GeoCities_Rebuild.sha256 > ~/Desktop/GeoCities_Rebuild.sha256.status1
# Look for errors in status1 file

```

Then we would run it again.

 

```
# And again on the HPE RAID after transfer
cd /run/user/1000/gvfs/smb-share:server=truenas.local,share=archived%20websites/GeoCities/Rebuild_v5
sha256sum -c GeoCities_Rebuild.sha256 > ~/Desktop/GeoCities_Rebuild.sha256.status2
# Look for errors in status2 file

```

We can then join all the files to the giant single tar file.

 

```
# If constructed, it will make a giant 793.9 GB (793,889,955,840 bytes) tar archive.
# Direct cat GeoCities_Rebuild.tar.* to CHONK (otherwise space will run out)
cd /media/ubuntu/GC_2TB/REBUILD/
cat GeoCities_Rebuild.tar.* > /media/ubuntu/CHONK/GeoCities_Rebuild.tar

```

 

```
# Then verify that the archiving is OK with 7z
7z t /media/ubuntu/CHONK/GeoCities_Rebuild.tar

```
